Ping Pong no Scratch

Neste tutorial, vamos aprender a como criar um projeto de ping-pong no scratch.

1- Criando a Plataforma

  • Apagar o Ator que vem de fábrica.
  • Selecionar um novo Ator.
  • Montar um código para mover a plataforma

1.1- Apague o Ator que vem de fábrica

A não ser que você queira usar o gatinho como sua plataforma, ele não será útil para nós. Então vamos deletá-lo

1.2- Selecione um novo Ator

Agora, clique no botão "Selecione um Ator" para escolhermos o Ator que servirá como nossa plataforma.

1.3- Programando o movimento

Com o ator selecionado, vamos começar a programar a movimentação.
